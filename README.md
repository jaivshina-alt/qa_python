Написано 11 автотестов:

1) Проверка, что метод add_new_book не добавляет в словарь books_genre книгу с длинным названием (более 40 символов).
2) Проверка, что метод add_new_book не добавляет в словарь books_genre дубликат уже добавленной книги.
3) Проверка, что метод set_book_genre устанавливает книге жанр из списка genre, а метод get_book_genre выводит жанр книги по её имени.
4) Проверка, что метод set_book_genre не устанавливает книге жанр не из списка genre, а метод get_book_genre выводит жанр книги по её имени.
5) Проверка, что метод get_books_with_specific_genre выводит список книг с определённым жанром.
6) Проверка, что метод get_books_genre выводит текущий словарь books_genre.
7) Проверка, что метод get_books_for_children возвращает книги, которые подходят детям (т.е. книги с жанрами из списка genre_age_rating не выводятся).
8) Проверка, что метод add_book_in_favorites добавляет книгу в избранное, а метод get_list_of_favorites_books выводит список избранных книг. 
9) Проверка, что метод delete_book_from_favorites удаляет книгу из избранного, а метод get_list_of_favorites_books выводит список избранных книг. 
10) Проверка, что метод add_book_in_favorites не добавляет в избраное книгу, которой нет в словаре books_genre, а метод get_list_of_favorites_books выводит список избранных книг. 
11) Проверка, что метод add_book_in_favorites не добавляет в избраное дубликат уже добавленной книги, а метод get_list_of_favorites_books выводит список избранных книг.
